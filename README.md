# SCT-Net

This repository provides supplementary materials for the manuscript:

**Road-Structure-Guided Sampling-Controlled Trajectory Aggregation for Road Extraction from High-Resolution Remote Sensing Imagery**

SCT-Net is designed for road extraction from high-resolution remote sensing imagery. It aims to improve road continuity under local road-cue insufficiency by organizing long-range road evidence along road-structure-guided sampling trajectories.

## Repository contents

- `DATA_AVAILABILITY.md`: data source information and access instructions for the public datasets used in the manuscript.
- `paper_visualizations/`: visualization results used in the manuscript.
- `results/`: quantitative result summaries supporting the reported experiments.
- `README.md`: description of the repository contents and availability statement.

The complete source code and trained model weights are not publicly released in this repository at the submission stage. Additional implementation details may be made available upon reasonable request to the corresponding author.
