# Data Availability

The raw datasets used in this study are publicly available from their original providers:

1. DeepGlobe Road Extraction Dataset
   http://deepglobe.org/

2. CHN6-CUG Roads Dataset
   https://github.com/CUG-URS/CHN6-CUG-Roads-Dataset

3. Massachusetts Roads Dataset
   https://www.cs.toronto.edu/~vmnih/data/

The raw images and annotations are third-party datasets. Therefore, this repository does not redistribute the raw datasets. Users should obtain the datasets from the original providers and comply with their corresponding terms of use.

This repository provides the source code and supplementary materials used to support the findings of the manuscript.
